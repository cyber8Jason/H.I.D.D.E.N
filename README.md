# H.I.D.D.E.N

Psychological Survival Horror built with Unreal Engine 5.4.4

## 🎮 Concept
H.I.D.D.E.N is a first-person survival horror game.
The game takes place in a secret underground bio-laboratory where human experiments have gone wrong.

The player cannot fight.
Survival depends on stealth, hiding, exploration and environmental interaction.

## 🧠 Core Themes
- Psychological Horror
- Isolation
- Vulnerability
- Lack of Control
- Blood and Gore

## ⚙ Tech Stack
- Unreal Engine 5.4.4
- Git + Git LFS
- Steam (planned release)
